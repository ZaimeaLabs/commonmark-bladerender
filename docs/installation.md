## Instalation

You can install the package via composer:

```bash
composer require zaimea/commonmark-bladerender
```

or via composer.json

```json
"repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/zaimealabs/commonmark-bladerender"
        }
    ]
```
